mp4 editor idea

extracts frames from an mp4 file using mp4box.js and allows some editing of the frames (e.g. image filters). more to come hopefully :)

NOTE: This has not been tested yet on mp4 files longer than a minute.

Additionally, when saving edits, the FPS is hardcoded to 120, image quality unfortunately is not preserved (so it may look worse than the source file) and any audio tracks are not included in the output.

instructions: click on a frame after loading an mp4 file to edit it. click on 'save changes' to export a new mp4 file with your edits.
